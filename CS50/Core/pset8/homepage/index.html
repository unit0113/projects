Project Overview

My goal is to create a python program which will conduct data gathering and analysis for the purposes of doing basic Due Dilligence (DD) on stocks. The current plan is for the program to have three possible outputs:

  • Output a visualization of important annual and quarterly data
  • Output a .csv file for easy copy/past into my own financial dashboard
  • Enable a comparison between two stocks, visualizing the same data as the first option, but adding in over/under data inbetween the two

The insperation for this project came from a user on reddit who posted their own attempt at this. Their examples (at least the ones that I ran across) look like this

Reference Images

Goals

  1. Have everything be in a .exe to make running easy
  2. Open with simple UI. This should allow the user to select between a single stock analysis and a comparison. If selecting a single stock, give the user the option to output relevent data in a .csv or some other file type that can be imported into excel
  3. Web scrape EDGAR (SEC filings database) for relevant annual and quarterly reports
  4. Display the data in useful charts
  5. Don't spend six months trying to figure out how to do this
