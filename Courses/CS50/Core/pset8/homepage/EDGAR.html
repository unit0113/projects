EDGAR Walkthrough

The program will need to navigate to the EDGAR website and insert the ticker. Alternatively, it could utilize the company's CIK number to navigate directly to the filings page, but the program will need a method of pairing ticker with CIK

From here, it will need to open the 10k's from the "10-K (annual reports) and 10-Q (quarterly reports)" section

Then it will need to enable the "View all 10-K's and 10-Q's" option in order to access the filings

Finally it will need to iterate over all of the filings, up to a certain date, and import that data in a useable form

(It sounds so easy now...)
